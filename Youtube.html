Introduction

Who Should Run the Ads?

The head of marketing wants to identify the top YouTubers in Poland in 2024 to determine which YouTuber(s) would be best suited for marketing campaigns.

Objectives

Key Pain Points

  • Identifying the top-performing YouTubers for marketing collaboration.
  • Ensuring data-driven decision-making to maximize campaign effectiveness.

Ideal Solution

To create a dashboard providing insights into the top YouTubers in Poland, including:

  • Subscriber count
  • Total views
  • Total videos
  • Engagement metrics

User Story

  • As the Head of Marketing, I want to use a dashboard analyzing YouTube channel data in Poland.
  • The dashboard should highlight top-performing channels based on metrics like subscriber base and average views.
  • This will enable me to make informed decisions on influencer collaborations for maximum impact.

Data Collection

Data Requirements

The required data includes:

  • Channel names
  • Total subscribers
  • Total views
  • Total videos uploaded

Data Sources

  • Kaggle
  • YouTube API

The dashboard should answer:

  • Who are the top 10 YouTubers with the most subscribers?

import numpy as np
                import pandas as pd
                from googleapiclient.discovery import build
                API_VERSION = 'v3'
                youtube = build('youtube', API_VERSION, developerKey=API_KEY)
                
                
                # Function to extract youtube metrics
                def get_channel_stats(youtube, channel_id):
                    request = youtube.channels().list(
                        part='snippet, statistics',
                        id=channel_id
                    )
                    response = request.execute()
                
                
                    if 'items' in response and response['items']:
                
                
                        data = dict(channel_name=response['items'][0]['snippet']['title'],
                                    total_subscribers=response['items'][0]['statistics']['subscriberCount'],
                                    total_views=response['items'][0]['statistics']['viewCount'],
                                    total_videos=response['items'][0]['statistics']['videoCount'],
                        )
                
                
                        return data
                    else:
                        return None
                
                
                # Read CSV into dataframe
                df = pd.read_csv("youtube_data_poland.csv")
                
                
                # Extract channel IDs and remove potential duplicates
                channel_ids = df['NAME'].str.split('@').str[-1].unique()
                
                
                # Initialize a list to keep track of channel stats
                channel_stats = []
                # keep track of channels returning none.
                No_stats = []
                
                
                # Loop over the channel IDs and get stats for each and keep track of channels with no stat.
                for channel_id in channel_ids:
                    stats = get_channel_stats(youtube, channel_id)
                    if stats is not None:
                        channel_stats.append(stats)
                    else:
                        No_stats.append(channel_id)
                
                
                # Convert the list of stats to a df
                stats_df = pd.DataFrame(channel_stats)

SQL Query


            /*
            # 1. Define variables
            # 2. Create a CTE that rounds the average views per video
            # 3. Select the columns you need and create calculated columns from existing ones
            # 4. Filter results by YouTube channels
            # 5. Sort results by net profits (from highest to lowest)
            */
            
            
            
            
            -- 1.
            DECLARE @conversionRate FLOAT = 0.02;           -- The conversion rate @ 2%
            DECLARE @productCost FLOAT = 20.0;               -- The product cost @ 20 PLN
            DECLARE @campaignCostPerVideo FLOAT = 10000.0;   -- The campaign cost per video @ 10,000 PLN
            DECLARE @numberOfVideos INT = 10;               -- The number of videos (10)
            
            
            
            
            -- 2.
            WITH ChannelData AS (
                SELECT
                    channel_name,
                    total_views,
                    total_videos,
                    ROUND((CAST(total_views AS FLOAT) / total_videos), -4) AS rounded_avg_views_per_video
                FROM
                    PoleYoutube
            )
            
            
            
            
            -- 3.
            SELECT
                channel_name,
                rounded_avg_views_per_video,
                (rounded_avg_views_per_video * @conversionRate) AS potential_units_sold_per_video,
                (rounded_avg_views_per_video * @conversionRate * @productCost) AS potential_revenue_per_video,
                ((rounded_avg_views_per_video * @conversionRate * @productCost) - (@campaignCostPerVideo * @numberOfVideos)) AS net_profit
            FROM
                ChannelData
            
            
            
            
            -- 4.
            WHERE
                channel_name IN ('tvnpl', 'TVN Series', 'Polsat')
            
            
            
            
            -- 5.
            ORDER BY
                net_profit DESC;

SQL Query


                
            /*
            # 1. Define variables
            # 2. Create a CTE that rounds the average views per video
            # 3. Select the columns you need and create calculated columns from existing ones
            # 4. Filter results by YouTube channels
            # 5. Sort results by net profits (from highest to lowest)
            */
            
            -- 1.
            DECLARE @conversionRate FLOAT = 0.02;        -- The conversion rate @ 2%
            DECLARE @productCost MONEY = 20.0;            -- The product cost @ 20 PLN
            DECLARE @campaignCost MONEY = 200000.0;      -- The campaign cost @ 200,000 PLN
            
            
            -- 2.
            WITH ChannelData AS (
                SELECT
                    channel_name,
                    total_views,
                    total_videos,
                    ROUND(CAST(total_views AS FLOAT) / total_videos, -4) AS avg_views_per_video
                FROM
                    PoleYoutube
            )
            
            -- 3.
            SELECT
                channel_name,
                avg_views_per_video,
                (avg_views_per_video * @conversionRate) AS potential_units_sold_per_video,
                (avg_views_per_video * @conversionRate * @productCost) AS potential_revenue_per_video,
                (avg_views_per_video * @conversionRate * @productCost) - @campaignCost AS net_profit
            FROM
                ChannelData
            
            
            -- 4.
            WHERE
                channel_name IN ('SBM', 'Step Records', N'Śpiewające Brzdące - Piosenki dla dzieci')
            
            
            -- 5.
            ORDER BY
                net_profit DESC;

  • Entertainment and Music channels drive the most engagement.
  • Kids' content dominates the top YouTube channels in Poland.

  • Caution with TV Networks: tvnpl, TVN Series, and Polsat post regularly but may not yield high returns.

  • Maximizing Reach: SBM stands out as an excellent channel for maximizing reach, consistently ranking among the top 10 in subscriber count and boasting high engagement with its content. Similarly, Step Records is alo a great option as their videos enjoy frequent engagement and as well have a large subscriber base.
